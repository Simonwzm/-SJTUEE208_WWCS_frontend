　　在 7 月 15 日举办的以“走进web3.0，探索数字经济发展新机遇”为主题的培训研讨会上，来自央视总台、新华网、光明网、中国网、法治日报、人民政协报、中国经济网、环球时报等30多家媒体单位的人士前来交流学习。　　欧科云链研究院高级研究员孙宇林向在场学员分享了《区块链产业和Web3.0发展现状及趋势》的课程，并从国家标准、科研实力、基础设施建设、行业应用等多个层面分析了当前国内区块链产业的发展形式。此外，孙宇林还通过重点讲解Web3.0的演变特征，论证了“向下一代互联网的集体迁徙已经开始”，现场媒体学员也带着对行业的好奇和兴趣纷纷加入了讨论中。　　研究机构 Messari 将 Web 1.0 到Web3.0进行了概括，如果说Web1.0仅为“可读”，互联网用户只能被动地浏览网页上的文本、图像以及简单的视频内容，那么Web 2.0 则为“可读 + 可写”，这一阶段用户不仅可以通过互联网平台获取信息，还可以自主创造内容并在传播原创内容的同时与其他用户进行互动。最新概念 Web 3.0由此诞生，被概括为“可读+可写+拥有”。孙宇林认为，“新一轮的 Web 3.0 思潮，如何让用户在数据处理和身份控制等方面拥有更大的自主权是核心，而应用的分布式是基本要求”。　　今年3月，中国证监会科技监管局局长姚前就撰文指出，“Web3.0是用户与建设者拥有并信任的互联网基础设施”，同时，“Web3.0是用户与建设者共建共享的新型经济系统”。　　“Web3.0不仅赋予用户自主管理身份，而且打破了中心化模式下数据控制者对数据的天然垄断。分布式账本技术可提供一种全新的自主可控数据隐私保护方案。用户数据经密码算法保护后在分布式账本上存储。身份信息与谁共享、作何种用途均由用户决定，只有经用户签名授权的个人数据才能被合法使用。通过数据的全生命周期确权，数据主体的知情同意权、访问权、拒绝权、可携权、删除权(被遗忘权)、更正权、持续控制权得到更有效的保障。” 　　在姚前的表述中，分布式账本技术被提及多次，这一技术在一些场景下也被称为区块链技术。为了让用户能够在 Web 3.0 的互联网上获得更多自主权，区块链技术(分布式账本技术)不可或缺。　　孙宇林表示，随着 Web 3.0 应用场景的丰富以及生态的逐渐完善，区块链上的数据也会种类越来越丰富、数量也会越来越多，大量的数据也将通过区块链进行固化流转，链上数据治理的重要性也因此将逐渐凸显。　　与Web2.0时代的互联网相似，以区块链为底层的Web3.0时代，数字化空间的安全仍然需要依赖对大量数据的处理与分析，其中就包括了通过数据判断安全事件主体的身份及行为，而这里我们所说的数据，将有相当一部分是存储在区块链上的。　　以目前区块链链上数据分析的实践经验来看，对链上地址进行标签化处理是比较普遍的做法，标签化也是大数据处理的一般做法。区块链地址标签分析在Web 3.0安全方面所能起到的作用也十分显著，比如对安全事件所涉及的恶意地址的资金流向进行分析，又或者针对某些特定地址的资金或资产的异常流动进行预警。　　目前，在数字资产交易领域，国内外的区块链数据分析已经比较成熟，特别是在数字资产安全事件发生后联合执法机构的资金追溯以及反洗钱等领域，区块链数据分析发挥了比较大的作用，在Web3.0时代，相关的数据分析经验也可以顺畅迁移。　　具体来说，区块链地址就如同银行卡号一样，是交易过程中最为重要的信息之一。而与传统银行卡号不同的是，任何一个用户都可以随意生成“无数”个地址，只要他持有私钥就可以随意构造并发起一次区块链交易。而这一过程虽然可以通过公开的区块链数据获取到，但也仅仅能看到地址之间的交易，无法进一步获知更多与交易主体相关的信息。　　区块链地址标签，就是为了在一定程度上通过数据体现出的规律和模式，尝试解决类似交易主体身份无法明确等问题。简单来说，地址标签就是某些地址所具备的一切属性。如果给一个人打标签的话，可以说这个人高、矮、胖、瘦、爱看书、爱旅游、脾气好等，链上地址也具有类似的属性，比如地址的格式、留存的资产、交易记录、交易时段等，就是地址的属性标签。通过地址标签，可以在某种程度上降低区块链的匿名性，提升对交易特征的识别度。　　区块链地址标签实际上是一个地址的各类属性，所以在分类上并没有一个统一的标准，例如，下图就是以太坊浏览器 etherscan.io 上支持的地址标签云词图 　　按地址标签属性的不同，可以从中挖掘出不同的价值和信息，对于地址标签，更多的是进行链上追踪时候的组合使用。　　以欧科云链链上天眼Pro的分析为例，如上图所示，通过对Silk Road 暗网中已添加标签地址的链上交易行为进行追踪，最终可以看到有部分资金流入了某些“中心化”数字资产交易所的地址。而数字资产交易所因其业务特性，可以在一定程度上完成对涉安全事件交易的定位和流向控制，可以理解为用户为了得到中心化平台服务提供的“便利”，必须提供一些个人信息来完成在平台的整个注册和认证过程。　　随着各类信息技术的迭代创新，数据作为战略资源的地位逐渐凸显。而在基于区块链的Web 3.0时代，链上数据的价值，除了地址标签，还需要通过一整套完整的解决方案，对链上数据进行检查、编目和解释，从而获得更具可执行性的洞察及见解。欧科云链通过自身多年积累的链上数据结构化处理经验，将积极赋能 Web 3.0 数据治理，为 Web 3.0 生态安全保驾护航。　　(以上图片由欧科云链授权中国网财经使用)
