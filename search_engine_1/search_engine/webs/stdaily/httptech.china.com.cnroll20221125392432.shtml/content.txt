　　近日，蚂蚁集团数字科技业务在菲律宾马尼拉举办客户交流会，与会嘉宾包括菲律宾银行营销协会 BMAP、东南亚最大投资银行CIMB、菲律宾首都银行Metrobank等机构的产品技术专家。蚂蚁数字科技业务在会上发布多款面向国际市场的新型数字化科技产品和解决方案。蚂蚁数科解决方案架构师张毅分享蚂蚁数字科技服务　　公开资料显示，随着东南亚数字化发展，东南亚购物、金融类移动app用户增长迅速，2022年DAU相比2021年增幅超20%。菲律宾政府在推进传统银行数字化转型的同时，积极促进数字银行的成立和运营。菲律宾数字化技术的采购需求旺盛，数字科技服务具有较大的发展前景。在交流会上，蚂蚁数科解决方案架构师张毅表示，感谢合作伙伴过去的支持与认可，蚂蚁非常看好菲律宾市场，希望持续为合作伙伴提供更安全、更方便、更可靠的服务。随后，他分享了一系列数字化科技产品和新功能。　　ZOLOZ Real ID eKYC(可信身份认证)　　基于支付宝实名认证安全体系发展起来的ZOLOZ Real ID eKYC(可信身份认证)服务，现已支持全球超过14个国家的在线身份认证需求，与东南亚国家传统的认证手段相比，ZOLOZ Real ID eKYC的用户身份验证时间从几天减少到三分钟，验证成功率从65% 提高到90% 以上。　　同一人使用多个ID注册套取平台营销资金、篡改证件冒用他人身份、团伙刷单等问题，已成为当前互联网、金融机构的常见风险。张毅分享了ZOLOZ Real ID 的“ID Network”新功能，可为合作伙伴构建用户数据图谱，利用领先的人工智能技术，识别上述批量欺诈风险。　　为加快用户的认证效率，ZOLOZ Real ID 还结合手机端NFC设备，研发便捷的护照认证功能，可实现秒速认证用户身份。此外，张毅透露风险名单管理、行为验证码、端安全等新功能也将陆续上线。　　ZOLOZ Smart AML(智能反洗钱)　　蚂蚁数科还向东南亚金融机构提供反洗钱风控能力，包含名单筛查、交易监控、客户风险评级等功能，帮助机构排查交易风险，满足当地监管合规要求。其自动化风险筛查功能，可利用人工智能技术辅助人工验证，减少80%的重复评审，评审效率提升40%以上。　　mPaaS(低代码移动开发平台)　　mPaaS是支持移动app/小程序高效开发的一站式解决方案，可生成初始化代码并提供模块化开发模式以进行协作开发。张毅介绍，对于处在数字化转型期间的银行和金融机构而言，开发效率非常关键。mPaaS提供的3 套开发框架、20 个功能模块、100+ UI组件可支持合作伙伴简单快速地构建专属的超级APP。此外，mPaaS联合埃森哲发布《移动应用开发平台产品白皮书》，向行业分享其在低代码技术方向上的实践和思考。　　参与活动的菲律宾首都银行Metrobank数字战略负责人Johdel C. Ocampo表示：蚂蚁集团ZOLOZ拥有非常先进和复杂的技术能力，可以精确地识别用户身份，能做到客户体验与安全性之间的完美平衡。数字化的注册流程免去人工成本，让我们能够拥有更多渠道、获取更多客户，加快了市场拓展的速度。菲律宾的数字化产业正在高速发展，我们必须迎头赶上，这也是我们要部署很多蚂蚁集团提供的解决方案的原因。　　据了解，蚂蚁集团数字科技海外业务已为印尼、马来西亚、菲律宾、泰国等数十个国家和地区提供数字化转型过程中的风险防御、研发提效、便捷支付、跨行业协同等解决方案，已经覆盖金融、保险、证券、信贷、电信、公众服务等领域。　　(以上图片由蚂蚁集团授权中国网财经使用) 神舟十五号计划择机发射郑州机场北货运区试运营广州机务段“机车大夫”宁德助力新能源产业发展2021年全国两会聚焦“中国力量”2021年全国两会金融业报道2021年3·15消费曝光台中医名医汇 解读中医本源中小银行普惠金融之道对话金融科技二十人聚焦2020年商业银行年报金融护航六稳六保 构建双循环银行业分支机构服务实体经济巡礼大力发展信用卡 助力构建双循环第一届区块链创新高峰论坛中国新经济企业500强发布会2020中国国际金融安全论坛2020年外滩金融峰会第三届中国医养保险高峰论坛区块链大航海时代行业峰会中国网是国务院新闻办公室领导，中国外文出版发行事业局管理的国家重点新闻网站。本网通过10个语种11个文版，24小时对外发布信息，是中国进行国际传播、信息交流的重要窗口。凡本网站注明“来源：中国网科技”的所有作品，均为本网合法拥有版权或有权使用的作品，未经本网授权不得转载、摘编或利用其它方式使用上述作品。电话：0086-10-82081166-6059传真：0086-10-82081900邮箱：alltech@china.org.cn官方微信官方微博
